SuperMap.Layer.Graph

统计专题图。

统计专题图通过为每个要素绘制统计图表来反映其对应的专题值的大小。它可同时表示多个字段属性信息，在区域本身与各区域之间形成横向和纵向的对比。 统计专题图多用于具有相关数量特征的地图上，比如表示不同地区多年的粮食产量、GDP、人口等，不同时段客运量、地铁流量等。允许一次分析多个数值型变量， 即可以将多个变量的值绘制在一个统计图上。目前提供的统计图类型有：柱状图(Bar)，折线图(Line)，饼图(Pie)。

1.chartsSetting 基础属性

width {Number} 专题要素（图表）宽度，必设参数。
height {Number} 专题要素（图表）高度，必设参数。
codomain {Array{Number}} 值域，长度为 2 的一维数组，第一个元素表示值域下限，第二个元素表示值域上限，必设参数。
XOffset {Number} （所有）专题要素（图表）在 X 方向上的偏移值，单位像素。
YOffset {Number} （所有）专题要素（图表）在 Y 方向上的偏移值，单位像素。
dataViewBoxParameter {Array{Number}} 数据视图框参数，它是指图标框（由图标位置、图标宽度、图标高度构成的图表范围框）在左、下，右，上四个方向上的内偏距值，此参数只在 useDefaultDVBParameter 为 false 时生效。
useDefaultDVBParameter {Boolean} 始终使用图形类型的默认数据视图框参数。默认值：true。
decimalNumber {Number/Boolean} 小数位数(数据的小数位处理)，取值范围：[0, 16]。如果不设置此参数，不对数据做小数位处理。
axisXBlank {Array{Number}} x 轴上的空白参数。长度为 3 的数组，第一元素表示起始图形与 x 轴起点的间距，第二个元素表示图形间空白间距， 第三个元素表示最后一个图形与 x 轴末端的间距 。此参数对图表 bar，Line 有效。
dataStyle {Object} 数据的可视化对象 style，此参数控制数据的可视化对象的基础样式，优先级低于 dataStyleByFields，dataStyleByCodomain。
dataStyleByFields {Array{Object}} 数据的可视化对象 style，此参数按字段控制数据的可视化对象样式，优先级低于 dataStyleByCodomain，高于 dataStyle。 此参数中的 style 与 themeFields 中的字段一一对应 。例如： themeFields 为 [“POP_1992”, “POP_1995”, “POP_1999”], dataStyleByFields 为[style1, style2, style3], 则在图表中，字段 POP_1992 对应的可视化对象使用 style1，字段 POP_1995 对应的可视化对象使用 style2 ，字段 POP_1999 对应的可视化对象使用 style3。
dataStyleByCodomain {Array{Object}} 数据的可视化对象 style，按数据所在值域范围控制数据的可视化对象样式。

2.chartsSetting 坐标轴

useAxis {Boolean} 是否使用坐标轴。此参数对图表 Bar 有效，默认值：true。

从专题图中删除 feature。这个函数删除所有传递进来的矢量要素（数据）。 参数中的 features 数组中的每一项，必须是已经添加到当前图层中的 feature， 如果无法确定 feature 数组，则可以调用 removeAllFeatures 来删除所有feature。 如果要删除的 feature 数组中的元素特别多，推荐使用 removeAllFeatures， 删除所有feature后再重新添加。这样效率会更高。

destroy: function()
销毁图层，释放资源。调用此函数后，图层对象的所有属性将被字为空。
setChartsType: function( chartsType )
设置图表类型，此函数可动态改变图表类型。 在调用此函数前请通过 chartsSetting 为新类型的图表做相关配置。
addFeatures: function( features )
向专题图图层中添加数据 , 统计专题图仅接收 SuperMap.Feature.Vector 类型数据， feature 将储存于 features 属性中，其存储形式为数组。
removeFeatures: function( features )
从专题图中删除 feature。这个函数删除所有传递进来的矢量要素（数据）。 参数中的 features 数组中的每一项，必须是已经添加到当前图层中的 feature， 如果无法确定 feature 数组，则可以调用 removeAllFeatures 来删除所有feature。 如果要删除的 feature 数组中的元素特别多，推荐使用 removeAllFeatures， 删除所有feature后再重新添加。这样效率会更高。
removeAllFeatures: function()
清除当前图层所有的矢量要素。
redraw: function()
重绘该图层，成功则返回true，否则返回false。
clear: function()
清除图层。清除的内容包括数据（features） 、专题要素、缓存。
